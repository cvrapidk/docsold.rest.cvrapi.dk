CVR API REST API

Segmenterings API


Foretag søgning

Følgende endpoint skal bruges for at lave en segmenterings søgning: curl http://rest.cvrapi.dk/v1/dk/search/company



Parametre


life_name
Hele eller dele af et navn. F.eks.: Erhvervsstyrelsen

life_start
Virksomhedens startdato. Der er flere måde at søge på: år-måned-dag, år-måned, år-år. F.eks.: Søg virksomheder startet mellem 2010-2015 - eller virksomheder startet i oktober 2015: 2015-10.

life_end
Virksomhedens slutdato. Der er flere måde at søge på: år-måned-dag, år-måned, år-år. F.eks.: Søg virksomheder lukket mellem 2010-2015 - eller virksomheder lukket i oktober 2015: 2015-10.

contact_phone
Hele virksomhedens telefonnummer.

contact_email
Hele eller dele af virksomhedens e-mail adresse.

address_street
Hele eller dele af virksomhedens adresses vejnavn.

address_numberfrom
Virksomhedens adresses husnummer.

address_letterfrom
Virksomhedens adresses husnummer bogstav.

address_zipcode
Virksomhedens adresses postnummer. Kan angives som en range. F.eks.: 8000-9000.

address_city
Virksomhedens adresses bynavn.

address_municipality
Virksomhedens adresses kommunekode.

industrycode_text
Virksomhedens branchenavn.

industrycode_code
Virksomhedens branchekode. Kan angives som en range. F.eks.: 812410-821100.

industrycodes_text
Virksomhedens bibranchenavn.

industrycodes_code
Virksomhedens bibranchekode. Kan angives som en range. F.eks.: 812410-821100.

companyform_code
Virksomhedens virksomhedsformkode. Kan angives som en range. F.eks.: 80-81.

companyform_description
Virksomhedens virksomhedsform initialer. F.eks.: ApS, IVS, A/S.

companystatus_code
Virksomhedens statuskode. Se listen med statuskoder. Skriv koden fra statuskode-listen, for at søge på den bestemte status.

capital_capital
Virksomhedens kapital. Kan angives som en range. F.eks.: 1000-2500.

capital_currency
Virksomhedens kapitals valuta. F.eks.: DKK, EUR.

capital_ipo
Tjek om virksomheden er børsnoteret. Sættes til 'true'.

life_adprotected
Tjek om virksomheden er reklamebeskyttet. Sættes til 'true'.
